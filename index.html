Our Prototype

Our prototype Steam™ Machine is designed to play the latest AAA games in full 1080p. Using premium components and quiet fans this machine won't look out of place in the most stylish rooms, running almost silently and punching well above its weight.

Dual booting into both SteamOS™ and Windows 7™, this machine gives you the best of both worlds, with access to SteamOS™ content as well as games not yet available in the SteamOS™ library or the Steam™ platform.

Built with desktop components using the current Intel™ 1150 platform and utilising Nvidia's™ latest Maxwell architecture, this machine gives you multiple upgrade options for the future.
